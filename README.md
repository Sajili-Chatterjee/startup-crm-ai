# Startup CRM AI Platform

An AI-powered Startup CRM SaaS platform built using Next.js, Express.js, MongoDB, and OpenRouter AI APIs.

## Features

- JWT Authentication
- CRM Lead Management
- Dynamic Analytics Dashboard
- AI-Powered Startup Insights
- Conversion Funnel Analytics
- PDF Report Generation
- Protected Routes
- Responsive SaaS UI

## Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS
- Recharts
- shadcn/ui

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

### AI
- OpenRouter API

## Deployment

- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas
